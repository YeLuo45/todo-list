# TodoList V56 PRD: 智能优先级引擎 (Smart Priority AI)

## 1. 背景

V55 完成了 BatchOps + QueryAPI + Enhanced SmartSorter。V56 在此基础上实现**智能优先级引擎**，基于任务特征自动推断/推荐优先级，灵感来自 nanobot-design。

## 2. 方向

**Direction F: 智能优先级引擎** — 基于任务特征/逾期情况自动排序 + 智能推荐

## 3. 功能需求

### 3.1 SmartPriorityEngine 核心引擎
- `SmartPriorityEngine` 类：基于特征向量的优先级预测
- 特征向量：`[overdueDays, createdHours, tagCount, hasDeadline, repeatPattern, categoryMatch]`
- 权重可配置，支持运行时调整

### 3.2 优先级自动推断
- `predictPriority(task): { predicted: TaskPriority, confidence: number }`
- 基于特征自动推断缺失的 priority 字段
- `suggestPriority(task): TaskPriority` — 返回推荐优先级建议

### 3.3 批量优先级分配
- `autoPriorityTasks(tasks): Map<string, TaskPriority>`
- 扫描所有无明确 priority 的任务，自动分配
- 保留已有明确 priority 的任务

### 3.4 分数解释
- `explainScore(task): { score: number, breakdown: Record<string, number> }`
- 返回每个维度的贡献分，用于前端展示"为什么会排第一"

### 3.5 增强截止日期评分
- 距离现在 24h 内：紧急度最高（+50）
- 距离现在 48-72h：次紧急（+30）
- 超过 72h：线性递减
- 重复任务自动 boost（+20）
