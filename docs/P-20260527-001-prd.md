# TodoList V55 PRD: BatchOps + Query API + Enhanced SmartSorter

## 1. 背景

V54 完成了 MCP Retry + ToolCache。V55 在此基础上增强数据层批量操作和查询能力，以及 SmartSorter 的优先级算法扩展。

## 2. 方向

**Direction A 延续: BatchOps + Query API + Enhanced SmartSorter**

## 3. 功能需求

### 3.1 SQLiteStore Batch Operations
- `multiGet<T>(keys: string[]): Promise<Map<string, T | null>>`
- `multiPut(pairs: KvPair[]): Promise<void>`
- `multiDelete(keys: string[]): Promise<void>`
- `transaction(ops: BatchOp[]): Promise<BatchResult>`

### 3.2 SQLiteStore Query API
- `query<T>(filter: QueryFilter): Promise<T[]>`
- 支持 `where`, `orderBy`, `limit`, `offset`
- 内存索引加速常用查询（by-tag, by-status, by-priority）

### 3.3 SmartSorter Enhanced Features
- `sortByDeadline(tasks, beforeDate): Task[]` — 紧急优先排序
- `detectUnsortedZone(tasks): Task[]` — 识别"无归类"任务
- 权重可动态调整（运行时 updateWeights）
- 支持自定义评分函数 `customScorer`

## 4. 技术方案

```
src/db/
  sqlite-store.ts  — 添加 batch + query APIs
  index.ts        — 适配 NODE_ENV=test wasm loader

src/store/
  SmartSorter.ts  — 添加 deadline + unsorted zone + custom scorer
```
